# wiasliaw-claude-setup

Claude Code plugins and settings by wiasliaw.

## Installation

Run the following slash commands inside a Claude Code session:

### 1. Add Marketplace

```claude
/plugin marketplace add wiasliaw/wiasliaw-claude-setup
```

### 2. Install Plugin from Marketplace

```claude
/plugin install autoresearch@wiasliaw-wiasliaw-claude-setup
```
